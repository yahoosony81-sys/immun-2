# 노션 데이터베이스 설정 가이드

## ⚠️ 중요: 속성 이름 확인

코드에서 사용하는 노션 데이터베이스 속성 이름은 다음과 같습니다:

1. **이름** (Title 타입) - 기본 제공
2. **이메일** (Email 타입)
3. **전화번호** (Phone number 타입)
4. **희망 날짜** (Date 타입)
5. **희망 시간** (Select 타입) ⭐ 중요

## 희망 시간 Select 옵션 설정

노션 데이터베이스의 "희망 시간" 속성을 Select 타입으로 만들고, 다음 옵션들을 추가하세요:

### 옵션 목록 (오전 9시부터 오후 6시까지)

1. 09:00
2. 10:00
3. 11:00
4. 12:00
5. 13:00
6. 14:00
7. 15:00
8. 16:00
9. 17:00
10. 18:00

### 설정 방법

1. 노션 데이터베이스에서 "희망 시간" 속성 클릭
2. "Select" 타입 선택
3. "Add an option" 클릭하여 각 시간 옵션 추가
4. 정확히 위의 형식대로 입력 (예: "09:00", "10:00" 등)
5. 모든 옵션 추가 완료

## 속성 이름이 다른 경우

만약 노션 데이터베이스의 속성 이름이 위와 다르다면, 다음 파일들을 수정해야 합니다:

- `app/api/reservations/route.ts` - 예약 저장 시 사용
- `app/api/reservations/search/route.ts` - 예약 조회 시 사용
- `app/api/reservations/update/route.ts` - 예약 업데이트 시 사용

각 파일에서 속성 이름을 실제 노션 데이터베이스의 속성 이름과 일치시켜야 합니다.
