# 이미지 폴더

이 폴더에 히어로 섹션 배경 이미지를 넣어주세요.

## 사용 방법

1. 배경으로 사용할 이미지 파일을 이 폴더(`public/images/`)에 넣어주세요.
2. 파일 이름을 `hero-background.jpg` (또는 `.png`, `.webp` 등)로 저장하세요.
3. 또는 다른 이름을 사용하시려면 `components/HeroSection.tsx` 파일의 이미지 경로를 수정해주세요.

## 권장 사양

- 해상도: 1920x1080 이상 (고해상도 디스플레이 대응)
- 파일 형식: JPG, PNG, WebP
- 파일 크기: 500KB 이하 (로딩 속도 최적화)
