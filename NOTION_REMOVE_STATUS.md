# 노션 데이터베이스 "이체확인" 속성 삭제 가이드

## ⚠️ 중요 안내

코드에서 "이체확인" 속성을 사용하지 않도록 수정했습니다. 이제 노션 데이터베이스에서도 해당 속성을 삭제해야 합니다.

## 📋 노션 DB에서 "이체확인" 속성 삭제 방법

### 방법 1: 속성 숨기기 (권장 - 데이터 보존)

속성을 완전히 삭제하는 대신 숨길 수 있습니다. 나중에 필요하면 다시 표시할 수 있습니다.

1. 노션 데이터베이스 페이지 열기
2. "이체확인" 속성 열 헤더를 클릭
3. "Hide" 또는 "숨기기" 선택
4. 속성이 숨겨집니다 (데이터는 보존됨)

### 방법 2: 속성 완전 삭제

⚠️ **주의**: 이 방법은 속성과 모든 데이터를 영구적으로 삭제합니다. 되돌릴 수 없습니다!

1. 노션 데이터베이스 페이지 열기
2. "이체확인" 속성 열 헤더를 클릭
3. "Delete" 또는 "삭제" 선택
4. 확인 메시지에서 "Delete property" 또는 "속성 삭제" 클릭
5. 속성이 완전히 삭제됩니다

## ✅ 확인 사항

삭제 후 다음을 확인하세요:

- [ ] 노션 데이터베이스에서 "이체확인" 속성이 보이지 않음
- [ ] 예약을 새로 생성해도 "이체확인" 속성이 생성되지 않음
- [ ] 기존 예약 데이터는 그대로 유지됨
