# 노션 데이터베이스 "이체확인" 속성 추가 가이드

## "이체확인" 속성 추가하기

### 1단계: 노션 데이터베이스 열기
1. 노션에서 예약 데이터베이스 페이지 열기

### 2단계: 속성 추가
1. 데이터베이스 테이블에서 가장 오른쪽의 **"+"** 버튼 클릭
2. 또는 기존 속성 열 헤더를 클릭하여 **"Add a property"** 선택

### 3단계: 속성 타입 선택
1. 속성 이름 입력: **`이체확인`** (정확히 이 이름으로 입력)
2. 속성 타입 선택: **`Select`** (드롭다운 선택)
3. "Done" 또는 "완료" 클릭

### 4단계: Select 옵션 추가
1. "이체확인" 속성 열을 클릭하여 편집 모드로 진입
2. "Add an option" 또는 "옵션 추가" 클릭
3. 다음 2개의 옵션을 정확히 이 이름으로 추가:
   - **`이체전`**
   - **`이체완료`**

### 5단계: 기본값 설정 (선택사항)
1. "이체확인" 속성 설정에서 기본값을 **`이체전`**으로 설정할 수 있습니다

## 확인 사항

✅ 속성 이름이 정확히 **`이체확인`**인지 확인  
✅ 속성 타입이 **`Select`**인지 확인  
✅ 옵션이 정확히 **`이체전`**, **`이체완료`** 2개인지 확인  

## 주의사항

- 속성 이름은 대소문자를 구분하지 않지만, 정확히 **`이체확인`**으로 입력해야 합니다
- 옵션 이름도 정확히 **`이체전`**, **`이체완료`**로 입력해야 합니다
- 공백이나 특수문자가 포함되지 않도록 주의하세요

## 완료 후

속성을 추가한 후 예약을 다시 시도해보세요. 새로 생성되는 예약은 자동으로 "이체전" 상태로 설정됩니다.
